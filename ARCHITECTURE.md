# 系统架构设计文档

## 系统概述

轮胎AI模式识别系统采用模块化分层架构，实现从图像输入到质量评分的端到端自动化流程。

### 核心技术栈

- **编程语言**：Python 3.12
- **图像处理**：OpenCV, Pillow
- **数据处理**：NumPy
- **测试框架**：Pytest
- **日志系统**：Python logging

## 整体架构

```
┌─────────────────────────────────────────┐
│         服务层 (Service)                │
│  Preprocessor | Inference | Postproc   │
│  Scorer                                 │
├─────────────────────────────────────────┤
│         规则层 (Rules)                  │
│  rule1to5 | rule6_1 | rule6_2 | rule8_14 | ... │
├─────────────────────────────────────────┤
│         算法层 (Algorithms)             │
│  detection/     |    stitching/         │
│  (连续性/横沟检测) | (纵图/横图拼接)    │
├─────────────────────────────────────────┤
│         工具层 (Utility)                │
│  Logger | Exceptions | CV Utils | IO   │
├─────────────────────────────────────────┤
│         配置层 (Config)                 │
│  base | module | postprocessor | rules │
└─────────────────────────────────────────┘
```

## 核心模块

### 1. 预处理模块 (Preprocessor)

- 图像格式验证
- CMYK颜色空间转换
- 灰边去除
- 图像尺寸标准化
- 质量检查

### 2. 推理模块 (Inference)

**注意**：推理由外部模型完成，本项目读取模型输出结果。

- 轮胎花纹识别（外部模型）
- 中部区域花纹生成（外部模型）
- 边缘区域花纹生成（外部模型）
- 模式特征提取（外部模型）

### 3. 后处理模块 (Postprocessor)

实现完整的 9 阶段处理流程，使用规则层进行业务规则判定：

1. **Conf 处理**：配置加载和初始化
2. **小图筛选**：过滤小尺寸图片
3. **小图打分**：对小图进行初步评分
4. **纵图拼接**：纵向图片拼接（依赖 algorithms/stitching/vertical_stitch）
5. **横图拼接**：横向图片拼接（依赖 algorithms/stitching/horizontal_stitch）
6. **横图打分**：对横图进行评分
7. **装饰边框**：添加装饰边框
8. **统计总分**：计算综合得分
9. **整理输出**：结果整合和输出

### 4. 规则模块 (Rules)

实现业务规则判定，依赖算法层进行检测/拼接：

- **rule1to5**：规则 1～5
- **rule6_1**：规则 6-1
- **rule6_2**：规则 6-2
- **rule8_14**：规则 8 & 14
- **rule13**：规则 13
- **rule19**：规则 19

规则配置驱动，支持灵活配置。

### 5. 评分模块 (Scorer)

使用规则层进行评分计算：

- **业务规则评分**：基于规则层的业务判定
- **美感质量评分**：图像美感评估
- **综合评分计算**：整合各项得分

### 6. 工具模块 (Utils)

- **日志系统**：文件和控制台双输出，可配置日志级别
- **异常处理**：完整的异常层次结构（TireAIError基类）
- **图像处理**：图像读取/保存、格式转换、质量检查
- **IO工具**：文件读写、路径处理

## 数据结构

### 主要数据类

- **Task**: 任务数据结构
- **ImageData**: 图像数据结构
- **DetectionResult**: 检测结果数据结构
- **ContinuityResult**: 连续性检测结果

## 关键技术

### 1. 连续性检测算法

基于图像边缘端点的匹配算法：
- 自适应阈值计算（Otsu算法）
- 两种端点提取方法（纯像素操作/轮廓检测）
- 粗细线识别和匹配
- 匹配率计算和判定

### 2. 图像拼接技术

- 基于连续性检测的智能拼接
- 自动对齐算法
- 重叠区域优化

### 3. 横沟检测算法

基于卷积核的横沟识别：
- 横向粗线条检测（横沟）
- 纵向细线条交叉点识别
- RIB 区域自适应判定
- 横沟数量与交叉点数量评分

### 4. 性能优化

- NumPy向量化操作
- 内存预分配
- 批处理优化
- 并发处理支持

### 5. 异常处理机制

- 分层异常处理
- 自定义异常体系
- 错误恢复机制

## 数据流向

```
输入图像 → 预处理 → 推理* → 后处理 → 评分 → 输出结果
                         ↓        ↓
                       rules ←────┘
                         ↓
                    algorithms
                         ↓
                      工具层

* 推理：外部模型调用，模型输出存储到约定位置
```
